发表于2024-12-18|更新于2024-12-18
|浏览量:

Lombok失效

如果你使用Lombok注解声明，可以调用方法，但是出现
java:15:13 java: 找不到符号 符号: 方法 setDeptno(int) 位置: 类型为com.example.demo.domain.Dept的变量这种错误，大概率是因为lombok的注解没有被正确识别
首先你要安装lombok插件并且导入lombok依赖
IntelliJ IDEA
前往 File > Settings (或者 Ctrl+Alt+S) > Build, Execution, Deployment > Compiler > Annotation Processors。
确保勾选了 Enable annotation processing 和 Obtain processors from project classpath。
Eclipse
在 Eclipse 中，Lombok 安装脚本通常会自动配置注解处理器。如果你手动安装，请确保 Preferences > Java > Compiler > Annotation Processing 中的设置是正确的。
勾选完成后重构项目，就可以解决该问题。
